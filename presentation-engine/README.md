# CodeSlides — Интерактивные презентации по программированию

## Быстрый старт

1. Откройте `index.html` в браузере
2. Нажмите «Загрузить JSON файл» или перетащите файл в зону загрузки
3. Используйте стрелки для навигации

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `→` `↓` `Space` | Следующий слайд |
| `←` `↑` | Предыдущий слайд |
| `Home` | Первый слайд |
| `End` | Последний слайд |
| `T` | Содержание |
| `F` | Полный экран |
| `D` | Смена темы (светлая/тёмная) |
| `?` | Справка по клавишам |
| `Esc` | Закрыть панель |

## Формат JSON

### Верхний уровень

```json
{
  "meta": { ... },
  "slides": [ ... ]
}
```

### Метаданные (`meta`)

| Поле | Тип | Описание |
|------|-----|----------|
| `title` | string | **Обязательно.** Название презентации |
| `subtitle` | string | Подзаголовок (номер лекции и т.д.) |
| `description` | string | Краткое описание |
| `author` | string | Автор |
| `date` | string | Дата |
| `language` | string | Язык программирования: `kotlin`, `javascript`, `python` |
| `version` | string | Версия |
| `tags` | string[] | Теги |

### Типы слайдов

#### `title` — Титульный слайд

```json
{
  "type": "title",
  "title": "Название",
  "subtitle": "Подзаголовок",
  "author": "Автор",
  "date": "2024"
}
```

#### `section` — Разделитель секций

```json
{
  "type": "section",
  "sectionNumber": "Раздел 01",
  "title": "Название раздела"
}
```

#### `content` — Контентный слайд

```json
{
  "type": "content",
  "title": "Заголовок слайда",
  "reference": "https://example.com",
  "animate": true,
  "content": [
    { ... },
    { ... }
  ]
}
```

### Типы контентных блоков

#### `text` — Текст
```json
{ "type": "text", "text": "Поддерживает **жирный**, *курсив*, `код` и [ссылки](url)" }
```

#### `code` — Блок кода
```json
{
  "type": "code",
  "language": "kotlin",
  "filename": "Main.kt",
  "lineNumbers": true,
  "highlightLines": [2, 3],
  "code": "fun main() {\n    println(\"Hello\")\n}"
}
```

#### `list` — Маркированный список
```json
{ "type": "list", "items": ["Пункт 1", "Пункт 2", "Пункт 3"] }
```

#### `ordered-list` — Нумерованный список
```json
{ "type": "ordered-list", "items": ["Первый", "Второй", "Третий"] }
```

#### `callout` — Информационный блок
```json
{
  "type": "callout",
  "variant": "info | warning | error | success | tip",
  "title": "Заголовок",
  "text": "Текст"
}
```

#### `table` — Таблица
```json
{
  "type": "table",
  "headers": ["Col 1", "Col 2"],
  "rows": [
    ["A", "B"],
    ["C", "D"]
  ]
}
```

#### `columns` — Двухколоночная раскладка
```json
{
  "type": "columns",
  "ratio": "50-50 | 60-40 | 40-60",
  "columns": [
    [ { "type": "text", "text": "Left" } ],
    [ { "type": "text", "text": "Right" } ]
  ]
}
```

#### `comparison` — Сравнение
```json
{
  "type": "comparison",
  "goodLabel": "Правильно",
  "badLabel": "Неправильно",
  "good": "Хороший пример",
  "bad": "Плохой пример"
}
```

#### `steps` — Пошаговая инструкция
```json
{
  "type": "steps",
  "steps": [
    { "title": "Шаг 1", "text": "Описание" },
    { "title": "Шаг 2", "text": "Описание" }
  ]
}
```

#### `definition` — Определение
```json
{ "type": "definition", "term": "Термин", "description": "Описание" }
```

#### `quote` — Цитата
```json
{ "type": "quote", "text": "Текст цитаты", "author": "Автор" }
```

#### `formula` — Формула
```json
{ "type": "formula", "text": "E = mc²" }
```

#### `image` — Изображение
```json
{ "type": "image", "src": "url", "alt": "описание", "caption": "Подпись", "width": "80%" }
```

#### `link` — Ссылка
```json
{ "type": "link", "text": "Текст ссылки", "url": "https://example.com" }
```

#### `divider` — Разделитель
```json
{ "type": "divider" }
```

### Инлайн-форматирование

В текстовых полях (`text`, `title`, элементы списков и т.д.) поддерживается:

| Синтаксис | Результат |
|-----------|-----------|
| `**жирный**` | **жирный** |
| `*курсив*` | *курсив* |
| `` `код` `` | `код` |
| `[текст](url)` | ссылка |
| `\n` | перенос строки |

## Реализованные возможности

1. **Движок презентаций** — навигация клавиатурой, свайпы, колёсико мыши, анимации переходов
2. **16 типов контентных блоков** — текст, код, списки, таблицы, callout-блоки, колонки, сравнения, шаги, определения, цитаты, формулы, изображения, ссылки, разделители
3. **Подсветка синтаксиса** — встроенный хайлайтер для Kotlin, JavaScript, Python (без зависимостей)
4. **Тёмная/светлая темы** — переключение по кнопке или клавише `D`, сохранение в localStorage
5. **Полноэкранный режим** — с автоскрытием заголовка
6. **Оглавление** — боковая панель с навигацией по слайдам
7. **Загрузка JSON** — через кнопку или drag & drop
8. **Прогресс-бар и точки навигации** — визуальное отслеживание прогресса
9. **Копирование кода** — одной кнопкой
10. **Адаптивный дизайн** — работает на мобильных устройствах
11. **Стандартизированная JSON-схема** — справочный файл `presentations/schema.json`
